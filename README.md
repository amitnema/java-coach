# Java Coach
Java learning with different java versions

Currently, Java-Coach includes the changes for java7 and java8. It includes the tests for the respective versions.

# Build & Test

#### For all version included 
```sh
mvn clean test
```
#### For version 7
```sh
mvn clean test -Pjava7
```
